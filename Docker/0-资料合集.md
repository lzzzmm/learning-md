### UnionFS
UnionFS（Union File System）是一种文件系统，它允许将多个文件系统层叠在一起，使得用户和应用程序能够像访问单个文件系统一样访问这些层。UnionFS 最初设计用于 Linux 系统，广泛应用于虚拟化、容器化和一些操作系统的 Live CD。
