# CLR基础
## 1 CLR执行模型
CLR（Common Language Runtime， 公共语言运行时）是 .NET 框架的核心组件，它为 .NET 应用程序提供了一个运行时环境。CLR 扮演了虚拟机的角色，主要负责内存管理、程序集加载、安全性、异常处理和线程同步等功能。

CLR 是 .NET 平台独有的，而不是 C# 独有的。它是为整个 .NET 生态系统设计的运行时环境，不局限于任何特定语言。CLR 支持多种 .NET 语言，包括：
- C#
- VB\.NET
- F#
- 其他 .NET 兼容语言
